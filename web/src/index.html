How It Works

  1. Enter your seed phrase with ??? marking missing words
  2. Optionally provide a target address to verify against
  3. The tool generates all possible combinations
  4. Each combination is validated against BIP39 checksum
  5. Valid phrases are used to derive addresses
  6. Matching phrases are displayed
